# 业务判断
1. 按df_sheet_name判断作业类型判断：
    如果df_sheet_name包含"离职"、"减员"类似文字，{作业}='离职'
SELECT
    {作业} as '作业' ,
    ...
    改写为
SELECT
    '离职' as "作业",
    如果df_sheet_name包含"入职"、"增员"类似文字，{作业}='入职'
SELECT
    {作业} as '作业' ,
    ...
    改写为
SELECT
    '入职' as "作业",
2. 按信息项判断：
    必须首先检查df中存在的列名，并确保SELECT子句中引用的列都是实际存在的。
    如果"离职日期"或"入职日期"存在于df中：
SELECT
    {作业} as '作业' ,
    ...
    改写为
SELECT
    case 
        when not "离职日期" is null then '离职'
        when not "入职日期" is null then '入职'
        else ''
    end as "作业"
    否则改写为
SELECT
    null as "作业",
    ...
