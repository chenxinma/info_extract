# UI Demo

This directory contains a HTML demo for the Flask-based user interface for the info_extract project.

## Files

- `index.html`: The main HTML interface demo showing:
  - File upload functionality with drag & drop
  - Work directory configuration
  - Task management interface
  - Real-time log display
  - Processing status updates

## Features Demonstrated

- Modern UI with consistent color scheme (using primary color #165DFF)
- Responsive layout using Bootstrap
- File upload with validation
- Task management with status updates
- Real-time log display
- Processing simulation

The demo is a frontend-only implementation that showcases the planned UI functionality before backend integration.
